# 첫 번째 차트 만들기

5분 안에 환자를 등록하고 첫 차트를 완성하는 빠른 시작 가이드입니다.

---

## Step 1: 환자 등록

1. 상단 네비게이션(PC) 또는 하단 **"더보기"** 탭(모바일)에서 **"환자/보호자"** 페이지로 이동합니다.
2. **"+ 환자 등록"** 버튼을 클릭합니다.
3. 환자 정보를 입력합니다.
   - **동물 이름** (필수): 예) 초코
   - **종** (필수): 개, 고양이 등
   - **품종**: 예) 말티즈, 코리안숏헤어
   - **체중**: kg 단위 (차트의 약물 용량 계산에 사용됩니다)
   - **성별**: 수컷(M) / 암컷(F) / 중성화 수컷(MC) / 중성화 암컷(FS)
   - **나이**: 만 나이

[스크린샷: 환자 등록 모달 — 이름, 종, 품종, 체중, 성별, 나이 입력]

4. **"등록"** 버튼을 누르면 환자가 생성됩니다.

---

## Step 2: 보호자 등록

1. 같은 **"환자/보호자"** 페이지에서 **보호자** 탭을 선택합니다.
2. **"+ 보호자 등록"** 버튼을 클릭합니다.
3. 보호자 정보를 입력합니다.
   - **이름** (필수): 보호자 성명
   - **전화번호**: 연락 가능한 번호

[스크린샷: 보호자 등록 모달 — 이름, 전화번호 입력]

4. **"등록"** 버튼을 누르면 보호자가 생성됩니다.

---

## Step 3: 환자-보호자 연결

1. 등록한 환자 카드를 클릭하여 환자 상세 화면을 엽니다.
2. **"보호자 연결"** 영역에서 등록한 보호자를 검색하여 선택합니다.

> **팁:** 한 보호자에 여러 환자를 연결하거나, 한 환자에 여러 보호자를 연결할 수 있습니다.

---

## Step 4: 새 차트 생성

1. **차트 목록** 페이지로 이동합니다.
   - PC: 상단 네비게이션 탭
   - 모바일: 하단 1번째 탭 (차트 아이콘)

2. **"+" 버튼**을 클릭하여 차트 생성 모달을 엽니다.

3. 차트 정보를 입력합니다.
   - **환자 선택** (필수): 등록한 환자를 검색하여 선택
   - **날짜**: 기본값은 오늘 날짜
   - **Cage 번호**: 케이지/입원실 번호 (선택)
   - **담당 수의사**: 주치의 선택 (선택)

[스크린샷: 차트 생성 모달 — 환자 검색, 날짜, Cage#, 담당의 입력]

4. **"생성"** 버튼을 누르면 차트가 만들어지고, 자동으로 차트 상세 화면으로 이동합니다.

---

## Step 5: 기본 정보 입력 (고정행)

차트에 진입하면 **고정행**(Vital)이 미리 표시되어 있습니다. 체온, 심박수, 호흡수, 혈압 등 병원에서 정의한 기본 측정 항목입니다.

1. 차트 그리드에서 원하는 **시간대 열**의 셀을 클릭합니다.
2. 측정값을 입력합니다. 예) 체온: 38.5
3. 입력 후 다른 셀을 클릭하거나 Enter를 누르면 **자동으로 저장**됩니다.

[스크린샷: 차트 그리드 — 고정행(체온, 심박수 등)에 값이 입력된 모습]

> **팁:** 화살표 키(←→↑↓)나 Tab 키로 셀 사이를 빠르게 이동할 수 있습니다.
