# 병원 설정하기

회원가입 후 처음 로그인하면 **온보딩 화면**이 나타납니다. 새 병원을 직접 만들거나, 기존 병원에 합류할 수 있습니다.

---

## 온보딩 화면 구성

가입 직후 아래와 같은 선택 화면이 표시됩니다.

[스크린샷: 온보딩 선택 화면 — "기존 병원 합류"와 "새 병원 생성" 두 개의 카드]

- **기존 병원 합류**: 이미 Vet-Sync를 사용 중인 병원에 합류할 때
- **새 병원 생성**: 병원을 처음 등록할 때

---

## 옵션 1: 새 병원 만들기

처음 Vet-Sync를 도입하는 병원이라면 직접 병원을 생성합니다.

1. 온보딩 화면에서 **"새 병원 생성"** 카드를 선택합니다.

2. 병원 정보를 입력합니다.
   - **병원명** (필수): 예) ○○ 동물병원
   - **주소** (선택): 병원 소재지
   - **전화번호** (선택): 병원 연락처

[스크린샷: 병원 등록 폼 — 병원명, 주소, 전화번호 입력 필드]

3. **"시작하기"** 버튼을 누르면 병원이 생성되고, 바로 대시보드로 이동합니다.

> **팁:** 병원을 생성한 사람은 자동으로 **관리자(ADMIN)** 권한을 갖게 됩니다. 직원 초대와 병원 설정을 관리할 수 있습니다.

---

## 옵션 2: 기존 병원에 합류하기

이미 Vet-Sync를 사용 중인 병원에 합류하려면 관리자에게 **초대 코드**를 받아야 합니다.

### Step 1: 초대 코드 입력

1. 온보딩 화면에서 **"기존 병원 합류"** 카드를 선택합니다.
2. 관리자에게 받은 **6자리 초대 코드**를 입력합니다.
3. **"확인"** 버튼을 누릅니다.

[스크린샷: 초대 코드 입력 화면 — 6자리 코드 입력 필드]

### Step 2: 합류 확인

1. 초대 코드가 유효하면 **병원 이름**이 표시됩니다.
2. 맞는 병원인지 확인한 후 **"합류 요청"** 버튼을 누릅니다.

[스크린샷: 합류 확인 화면 — 병원명 표시 + "합류 요청" 버튼]

### Step 3: 승인 대기

1. 합류 요청을 보내면 **"승인 대기 중"** 화면으로 전환됩니다.
2. 병원 관리자가 요청을 승인하면 **자동으로 대시보드로 이동**합니다.

[스크린샷: 승인 대기 화면 — 스피너 + "관리자가 요청을 확인하면 자동으로 이동합니다" 메시지]

> **팁:** 승인 대기 화면에서 브라우저를 닫아도 괜찮습니다. 다시 접속하면 승인 상태를 자동으로 확인합니다.

> **주의:** 관리자가 요청을 **거절**하면 알림과 함께 다시 온보딩 화면으로 돌아갑니다. 올바른 초대 코드를 사용했는지 관리자에게 확인하세요.

---

## 초대 코드는 어디서 받나요?

병원 관리자가 **설정 → 병원 관리** 페이지에서 초대 코드를 확인하고 공유할 수 있습니다. 관리자에게 6자리 코드를 요청하세요.
