# 회원가입 & 로그인

Vet-Sync를 사용하려면 먼저 계정을 만들어야 합니다. 이메일과 비밀번호만으로 간편하게 가입할 수 있습니다.

---

## 회원가입

1. Vet-Sync 접속 화면에서 하단의 **"회원가입"** 링크를 클릭합니다.

![로그인 화면 하단 회원가입 링크](/images/screenshots/help/signup-link.png)

2. 회원가입 폼에 아래 정보를 입력합니다.
   - **이름**: 병원 내에서 표시될 본인 이름
   - **이메일**: 로그인에 사용할 이메일 주소
   - **비밀번호**: 8자 이상
   - **비밀번호 확인**: 동일한 비밀번호 재입력

![회원가입 폼](/images/screenshots/help/signup-form.png)

3. **"회원가입"** 버튼을 누르면 가입이 완료되고, 자동으로 **병원 설정 화면**(온보딩)으로 이동합니다.

> **팁:** 가입 후 바로 병원을 설정해야 차트와 업무 기능을 사용할 수 있습니다. 병원 설정 방법은 [병원 설정하기](./onboarding.md)를 참고하세요.

---

## 로그인

1. Vet-Sync 접속 화면에서 **이메일**과 **비밀번호**를 입력합니다.
2. **"로그인"** 버튼을 클릭합니다.

![로그인 화면](/images/screenshots/help/login-form.png)

> **팁:** 로그인 상태는 자동으로 유지됩니다. 브라우저를 닫았다 열어도 다시 로그인할 필요가 없습니다.

---

## 데모 모드로 체험하기

계정을 만들지 않고 Vet-Sync의 주요 기능을 미리 체험할 수 있습니다.

1. 데모 사이트에 접속하면 **자동으로 체험 계정에 로그인**됩니다.
2. 샘플 환자, 차트, 업무 데이터가 미리 준비되어 있어 실제 사용 환경을 체험할 수 있습니다.
3. 데모 환경에서 입력한 데이터는 일정 시간 후 초기화됩니다.

> **주의:** 데모 모드는 기능 체험 용도이며, 실제 진료 데이터를 저장하는 용도로 사용하지 마세요.

---

## 비밀번호 변경

1. 로그인 후 **설정** 페이지로 이동합니다.
   - PC: 상단 헤더의 설정 아이콘
   - 모바일: 하단 탭 **"더보기"** → **"설정"**
2. **"비밀번호 변경"** 항목에서 현재 비밀번호와 새 비밀번호를 입력합니다.
3. **"변경"** 버튼을 누르면 비밀번호가 즉시 변경됩니다.

![비밀번호 변경](/images/screenshots/help/settings-password-change.png)

> **주의:** 비밀번호 변경 후에는 다른 기기에서 다시 로그인해야 할 수 있습니다.
