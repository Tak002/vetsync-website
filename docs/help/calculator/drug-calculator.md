# 약물 용량 계산기

차트에 등록된 약물 제품의 투여량을 환자 체중 기반으로 자동 계산합니다. 수동 계산 없이 정확한 용량을 처치행에 바로 반영할 수 있습니다.

## 계산기 열기

약물 용량 계산기는 **차트 상세 화면**에서 접근합니다.

1. 차트 목록에서 대상 차트를 선택합니다.
2. 차트 상세 화면 우측(PC) 또는 하단(모바일)의 **계산기 버튼**을 탭합니다.
3. PC에서는 차트 옆에 사이드 패널로, 모바일에서는 바텀 시트로 열립니다.

![약물 계산기 버튼](/images/screenshots/help/calc-drug-btn.png)

> **팁:** 계산기 패널 상단에 환자 체중이 표시됩니다. "체중 미등록"이라면 먼저 환자 정보에서 체중을 입력해주세요.

## 사용 흐름

### 1단계: 약물 제품 검색 및 선택

1. 계산기 패널 상단의 **제품 검색** 입력란에 제품명을 입력합니다.
2. 병원에 등록된 제품 목록에서 원하는 제품을 선택합니다.
3. 선택된 제품이 계산기 목록에 추가되고, 동시에 차트의 빈 처치행에 자동 할당됩니다.

![약물 제품 검색](/images/screenshots/help/calc-drug-product-search.png)

> **팁:** 이미 차트 처치행에 제품이 할당되어 있다면 계산기를 열 때 자동으로 목록에 표시됩니다. 별도로 추가할 필요가 없습니다.

### 2단계: 환자 체중 확인

- 환자 정보에 등록된 체중이 **자동 반영**됩니다.
- 계산기 헤더에 `00.0kg` 형태로 표시됩니다.
- 체중이 변경되면 모든 약물의 계산 결과가 **즉시 재계산**됩니다.

### 3단계: 용량 지수 입력

1. 각 약물 항목의 **용량 입력란**에 원하는 용량 지수를 입력합니다.
2. 오른쪽 드롭다운에서 용량 단위를 선택합니다.
   - **mg/kg**: 밀리그램/킬로그램 (기본값, mg 농도 제품)
   - **mcg/kg**: 마이크로그램/킬로그램 (mcg 농도 제품)
   - **IU/kg**: 국제 단위/킬로그램 (IU 농도 제품)

![용량 입력](/images/screenshots/help/calc-drug-dose-input.png)

> **주의:** 용량 단위와 제품 농도 단위가 호환되지 않으면(예: IU/kg 단위로 mg 농도 제품 계산) 경고 메시지가 표시됩니다. 이 경우 최종 투여량이 계산되지 않습니다.

### 4단계: 자동 계산 결과 확인

용량 지수를 입력하면 아래 정보가 실시간으로 표시됩니다:

- **계산 수식**: `체중 x 용량지수 / 농도` 형태의 수식
- **최종 투여량**: 제품 단위(ml, 정 등)로 환산된 결과

예시: 체중 5kg 환자에게 세파졸린(100mg/ml) 22mg/kg 투여 시
```
5kg x 22mg/kg / 100mg/ml = 1.10 ml
```

![계산 결과](/images/screenshots/help/calc-drug-result.png)

### 5단계: 처치행에 적용

1. 각 약물 항목 우측의 **"적용"** 버튼을 탭하면 해당 처치행에 계산 결과가 반영됩니다.
2. 여러 약물을 한 번에 반영하려면 하단의 **"모두 적용 (N)"** 버튼을 탭합니다.

![적용 버튼](/images/screenshots/help/calc-drug-apply-btns.png)

> **팁:** 적용 시 처치행에 용량 지수(dose rate), 단위, 계산된 투여량, 계산 시점 체중이 함께 저장됩니다.

## 약물 제거

- 계산기 목록에서 약물을 제거하려면 해당 항목 우측의 **휴지통 아이콘**을 탭합니다.
- 제거하면 차트 처치행에서도 해당 제품 할당이 해제됩니다.

## 체중 변경 시 동작

환자 체중이 업데이트되면:
- 계산기를 다시 열 때 **새 체중**이 자동 반영됩니다.
- 모든 약물의 계산 결과가 새 체중 기준으로 재계산됩니다.
- 이전에 적용한 결과는 유지되므로, 필요하다면 다시 "적용"을 눌러 갱신해주세요.

## 약물 레퍼런스 정보

제품에 연결된 약물이 있다면 Plumb's Veterinary Drug Handbook(10th Edition) 기반의 성분 정보를 참고할 수 있습니다. 약물 제품 관리에서 제품에 약물(성분)을 연결하면 활용 가능합니다.

> **팁:** 약물 제품 등록 시 농도 정보(concentration_value, 단위)를 정확히 입력해야 계산 결과가 올바릅니다. 제품 관리는 설정 메뉴에서 할 수 있습니다.
