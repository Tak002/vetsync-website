# 수액 계산기

수액 베이스에 첨가 약물을 추가하고, 투여 속도와 총 수액량을 기반으로 약물 투입량을 자동 계산합니다. 계산 결과는 차트의 수액행에 바로 반영됩니다.

## 계산기 열기

수액 계산기는 **차트 상세 화면**에서 접근합니다.

1. 차트 목록에서 대상 차트를 선택합니다.
2. 차트 상세 화면에서 **수액 계산기 버튼**을 탭합니다.
3. PC에서는 차트 옆에 사이드 패널로, 모바일에서는 바텀 시트로 열립니다.

[스크린샷: 차트 상세 화면에서 수액 계산기 버튼 위치]

> **팁:** 수액 열이 여러 개인 경우, 계산기 상단에 열 선택 세그먼트가 표시됩니다. 원하는 열을 탭하여 전환할 수 있습니다.

## 사용 흐름

### 1단계: 수액 베이스 선택

1. "수액 베이스" 드롭다운에서 기본액을 선택합니다.
2. 병원에 등록된 수액 베이스 목록에서 선택할 수 있습니다.
   - 대표적인 수액: NS(생리식염수), LRS(링거젖산액), D5W(5% 포도당액) 등

[스크린샷: 수액 베이스 선택 드롭다운]

> **팁:** 수액 베이스 목록은 설정 메뉴에서 관리할 수 있습니다. 병원에서 자주 사용하는 수액을 미리 등록해두세요.

### 2단계: 투여 속도 입력

1. "속도 (mL/kg/hour)" 입력란에 수액 투여 속도를 입력합니다.
2. 체중 기반 시간당 투여량(mL/kg/hr) 단위입니다.

[스크린샷: 수액 속도 입력란]

### 3단계: 첨가 약물 추가 (선택사항)

수액에 약물을 추가해야 하는 경우:

1. "추가 약물" 검색란에 제품명을 입력합니다.
2. 병원에 등록된 제품 목록에서 원하는 약물을 선택합니다.
3. 추가된 약물 항목에서 **약물 투여 속도**를 입력합니다.
4. 투여 속도 단위를 선택합니다:
   - **mg/kg/hr**: 밀리그램/킬로그램/시간 (기본값)
   - **mcg/kg/min**: 마이크로그램/킬로그램/분

[스크린샷: 첨가 약물 추가 후 속도 입력하는 모습]

> **팁:** 여러 약물을 동시에 추가할 수 있습니다. 각 약물별로 독립적인 속도와 단위를 설정합니다.

### 4단계: 수액량 입력

1. "수액량 (mL)" 입력란에 총 수액량을 입력합니다.
2. 수액량을 입력하면 첨가 약물의 계산 결과가 표시됩니다.

[스크린샷: 수액량 입력란과 약물 계산 결과]

> **주의:** 첨가 약물이 있는 경우, 수액량을 입력해야만 약물의 필요 투입량이 계산됩니다.

### 5단계: 계산 결과 확인 및 적용

1. 각 약물의 계산 수식과 필요 투입량이 실시간으로 표시됩니다.
2. 하단의 **"적용"** 버튼을 탭하면 수액행에 계산 결과가 반영됩니다.

[스크린샷: 적용 버튼]

## 약물 계산 방식

수액 첨가 약물의 투입량은 다음과 같이 계산됩니다:

**mg/kg/hr 단위인 경우:**
```
약물속도(mg/kg/hr) / 수액속도(mL/kg/hr) x 수액량(mL) / 약물농도(mg/mL)
= 필요 투입량(mL)
```

**mcg/kg/min 단위인 경우:**
먼저 mg/kg/hr로 변환한 뒤 동일한 수식을 적용합니다.
```
약물속도(mcg/kg/min) x 60 / 1000 = mg/kg/hr
```

## 첨가 약물 제거

- 각 약물 항목 우측의 **휴지통 아이콘**을 탭하면 해당 약물이 목록에서 제거됩니다.

## 계산기 모드 해제

이미 계산기로 적용된 수액행의 경우:
- 계산기 하단에 **"해제"** 버튼이 표시됩니다.
- "해제"를 탭하면 계산기 모드가 비활성화되고 수액행이 초기 상태로 돌아갑니다.
- 값을 수정한 뒤 **"다시 적용"** 버튼으로 업데이트할 수 있습니다.

## 기존 수액 정보 덮어쓰기

수동으로 입력된 수액 정보가 있는 수액행에 계산기를 적용하면:
- "수액 정보 덮어쓰기" 확인 다이얼로그가 표시됩니다.
- "덮어쓰기"를 선택하면 기존 입력 내용이 계산기 결과로 대체됩니다.
- "취소"를 선택하면 기존 정보가 유지됩니다.

> **주의:** 덮어쓰기하면 기존 수동 입력 내용은 복구할 수 없습니다. 신중하게 결정해주세요.
