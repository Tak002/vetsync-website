# 차트 복사

입원 환자처럼 매일 비슷한 처치를 반복하는 경우, 이전 차트의 구성을 복사하여 빠르게 새 차트를 세팅할 수 있습니다.

---

## 차트 복사 기능 접근

차트 상세 화면의 헤더 액션 바에서 **"차트 복사"** 버튼을 클릭하면 드롭다운 메뉴가 열립니다.

![차트 복사 메뉴](/images/screenshots/help/chart-copy-confirm.png)

드롭다운 메뉴 항목:
- **차트 복사하기** — 현재 차트 번호를 클립보드에 복사
- **전날 차트 붙여넣기** — 전날 차트 내용을 현재 차트로 가져오기
- **차트 붙여넣기** — 특정 차트의 내용을 현재 차트로 가져오기

---

## 전날 차트 붙여넣기

같은 환자의 **어제 날짜 차트**가 있으면 버튼 한 번으로 내용을 가져올 수 있습니다.

1. **"차트 복사"** 드롭다운에서 **"전날 차트 붙여넣기"**를 선택합니다.
2. 확인 다이얼로그가 표시됩니다: "전날 차트를 불러오시겠습니까?"
3. **"불러오기"**를 누르면 전날 차트의 내용이 복사됩니다.

![차트 복사 버튼](/images/screenshots/help/chart-copy-btn.png)

> **주의:** 전날에 해당 환자의 차트가 없으면 이 옵션은 비활성화됩니다. "(차트 없음)"으로 표시됩니다.

---

## 특정 차트에서 붙여넣기

다른 날짜의 차트나 다른 환자의 차트에서도 내용을 가져올 수 있습니다.

### Step 1: 원본 차트 번호 복사
1. 복사하고 싶은 원본 차트의 상세 화면으로 이동합니다.
2. **"차트 복사"** → **"차트 복사하기"**를 선택하여 차트 번호를 클립보드에 복사합니다.

### Step 2: 대상 차트에 붙여넣기
1. 내용을 가져올 대상 차트의 상세 화면으로 이동합니다.
2. **"차트 복사"** → **"차트 붙여넣기"**를 선택합니다.
3. 클립보드의 차트 번호가 자동으로 읽히고, 원본 차트 정보가 표시됩니다.
4. 내용을 확인한 후 **"불러오기"**를 누릅니다.

![차트 붙여넣기 확인](/images/screenshots/help/chart-paste-confirm.png)

> **팁:** 확인 다이얼로그에서 원본 차트 정보를 클릭하면 해당 차트로 바로 이동할 수 있습니다.

> **팁:** 클립보드 접근 권한이 없는 브라우저에서는 차트 번호를 직접 입력할 수 있는 폼이 표시됩니다.

---

## 복사 대상

차트 복사 시 가져오는 항목과 가져오지 않는 항목:

| 구분 | 복사됨 | 복사 안 됨 |
|---|---|---|
| C.C. (주호소) | O | |
| 진단명 | O | |
| 처치행 구성 (이름, 순서) | O | |
| 수액행 구성 | O | |
| 마킹 설정 | O | |
| 셀 입력값 (시간대별 데이터) | | X |
| 완료 상태 | | X |

> **주의:** 복사하면 현재 차트의 C.C., 진단명, 처치행, 마킹 데이터가 **덮어씌워집니다**. 기존에 입력한 내용이 있다면 유실될 수 있으므로 주의하세요.
