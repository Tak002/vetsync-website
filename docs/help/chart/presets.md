# 차트 프리셋 사용

프리셋은 자주 사용하는 처치 조합을 **템플릿으로 저장**해두고, 새 차트에 한 번에 적용하는 기능입니다.

---

## 프리셋이란?

특정 질병이나 수술 후 반복적으로 사용하는 처치 항목들을 미리 묶어서 저장한 것입니다.

예시:
- **"중성화 수술 후"** 프리셋: Cefazolin, Meloxicam, Tramadol
- **"구토 관리"** 프리셋: Maropitant, Metoclopramide, Omeprazole
- **"일반 입원"** 프리셋: 기본 Vital 마킹 + Cefazolin

---

## 차트에 프리셋 적용하기

1. 차트 상세 화면의 헤더 액션 바에서 **"프리셋"** 버튼을 클릭합니다.

![프리셋 버튼](/images/screenshots/help/chart-preset-btn.png)

2. 등록된 프리셋 목록이 팝업으로 표시됩니다. 각 프리셋 이름 아래에 포함된 항목이 요약 표시됩니다.

![프리셋 목록 팝업](/images/screenshots/help/preset-list-popup.png)

3. 적용하려는 프리셋을 **클릭**합니다.

4. 프리셋에 포함된 고정행 마킹과 처치행이 **현재 차트에 일괄 추가**됩니다.

5. 적용 결과가 토스트 메시지로 표시됩니다.
   - 성공: `프리셋 "중성화 수술 후" 적용 완료 (3건)`
   - 일부 경고: `적용 완료 (2건) — 이미 존재하는 항목 제외`

> **팁:** 프리셋을 적용해도 기존에 입력한 처치행이 삭제되지는 않습니다. 새 항목이 추가되는 방식입니다.

---

## 프리셋 생성 & 관리

프리셋의 생성, 수정, 삭제는 **설정 페이지**에서 할 수 있습니다.

- PC: 상단 헤더 → 설정 아이콘 → 프리셋 관리
- 모바일: 하단 탭 "더보기" → "설정" → 프리셋 관리

### 프리셋 만들기
1. 설정 → 프리셋 관리에서 **"새 프리셋"** 버튼을 클릭합니다.
2. 프리셋 이름을 입력합니다. 예) "중성화 수술 후"
3. 포함할 항목을 추가합니다. 처치 항목명을 하나씩 입력합니다.
4. **"저장"**을 누르면 프리셋이 생성됩니다.

![프리셋 생성](/images/screenshots/help/preset-create-form.png)

> **팁:** 프리셋을 변경하면 같은 병원의 모든 스태프에게 실시간으로 반영됩니다. 별도의 새로고침이 필요 없습니다.

> **주의:** 프리셋 관리는 적절한 권한이 있는 스태프만 가능합니다.
