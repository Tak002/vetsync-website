# 고정행 (Vital) 사용법

고정행은 모든 차트에 기본으로 표시되는 측정 항목입니다. 체온, 심박수, 호흡수 등 Vital를 시간대별로 기록합니다.

---

## 고정행이란?

고정행은 병원에서 정의한 **기본 측정 항목**으로, 차트를 생성하면 자동으로 표시됩니다.

일반적인 고정행 예시:
- 체온 (BT)
- 심박수 (HR)
- 호흡수 (RR)
- 혈압 (BP)
- 체중 (BW)
- 구토
- 배변 / 배뇨
- 식이

![고정행 영역](/images/screenshots/help/chart-fixed-rows.png)

> **팁:** 고정행의 종류와 순서는 병원 관리자가 설정 페이지에서 커스터마이징할 수 있습니다.

---

## 값 입력 방법

1. 기록하려는 고정행과 시간대가 만나는 **셀을 클릭**합니다.
2. 측정값을 입력합니다.
   - 체온: `38.5`
   - 심박수: `120`
   - 혈압: `120/80`
3. **Enter**를 누르거나 다른 셀을 클릭하면 자동 저장됩니다.

![고정행 입력](/images/screenshots/help/chart-fixed-row-input.png)

---

## 빠른 입력이 지원되는 항목

일부 고정행(구토, 배변, 배뇨, 식이)에서는 **미리 정의된 선택지**가 팝업으로 표시됩니다. 한 번의 클릭으로 빠르게 입력할 수 있습니다.

| 항목 | 선택지 |
|---|---|
| 구토 | 위액 / 음식물 / 액체 / 거품 |
| 배변 | 정상 / 연변 / 설사 / 혈변 |
| 배뇨 | 정상 / 혈뇨 / 이상뇨 |
| 식이 | 정상 / 감소 / 결식 / 강제급여 |

![빠른 입력 팝업](/images/screenshots/help/chart-quick-input-popup.png)

> **팁:** 선택지 외에 직접 텍스트를 입력할 수도 있습니다.

---

## 체중 입력 시 자동 동기화

체중(BW) 고정행에 새 값을 입력하면, **환자 정보의 체중도 자동으로 업데이트**됩니다. 이를 통해 처치행의 약물 용량 계산이 최신 체중을 기반으로 이루어집니다.

> **팁:** 체중은 소수점 둘째 자리까지 입력할 수 있습니다. 예) 3.25 kg

---

## 고정행 커스터마이징

병원에서 사용하는 고정행의 종류와 순서를 변경하려면 **설정** 페이지에서 조정할 수 있습니다.

- PC: 상단 헤더 → 설정 아이콘
- 모바일: 하단 탭 "더보기" → "설정"

> **주의:** 고정행 설정 변경은 **관리자(ADMIN)** 권한이 필요합니다. 변경 사항은 이후 생성되는 차트에 적용됩니다.
