# 차트 생성 & 편집

차트는 환자의 하루 진료 기록 단위입니다. 환자 1명당 하루 1개의 차트를 생성하며, 고정행(Vital), 처치행(약물/시술), 수액행으로 구성됩니다.

---

## 새 차트 만들기

1. 차트 목록(대시보드) 화면에서 **"+" 버튼**을 클릭합니다.

[스크린샷: 차트 목록 우측 하단의 + 버튼]

2. 차트 생성 모달이 열리면 아래 항목을 입력합니다.

| 항목 | 필수 | 설명 |
|---|---|---|
| 환자 | 필수 | 환자 이름으로 검색하여 선택 |
| 날짜 | 필수 | 차트 날짜 (기본값: 오늘) |
| Cage# | 선택 | 케이지/입원실 번호 |
| 담당 수의사 | 선택 | 주치의 선택 |

[스크린샷: 차트 생성 모달 — 환자 선택, 날짜, Cage#, 담당의 입력 필드]

3. **"생성"** 버튼을 누르면 차트가 생성되고, 자동으로 차트 상세 화면으로 이동합니다.

> **주의:** 같은 환자에 대해 같은 날짜의 차트가 이미 존재하면 중복 생성이 불가합니다. "해당 날짜에 이미 차트가 존재합니다" 메시지가 표시됩니다.

---

## 차트 헤더 편집

차트 상세 화면 상단에는 환자 정보와 차트 메타 정보가 표시됩니다. 대부분의 필드는 **클릭하여 바로 수정**할 수 있습니다.

[스크린샷: 차트 헤더 — 환자명, Cage#, 주치의, C.C., 진단명이 표시된 모습]

### 편집 가능한 필드

| 필드 | 위치 | 편집 방법 |
|---|---|---|
| **C.C. (주호소)** | 헤더 2행 | 텍스트를 클릭하여 직접 입력 |
| **진단명** | 헤더 2행 | 텍스트를 클릭하여 직접 입력 |
| **담당 수의사** | 헤더 1행 우측 | 드롭다운에서 선택 |
| **Cage 번호** | 헤더 1행 | 뱃지를 클릭하여 수정 |

### 자동 저장

헤더의 모든 필드는 **자동 저장**됩니다. 입력이 완료되면 필드 옆에 저장 상태가 표시됩니다.

- **저장 중...**: 서버에 전송 중
- **저장됨** (체크 표시): 저장 완료
- **저장 실패** (경고 표시): 오류 발생 — 클릭하여 재시도

> **팁:** 차트 헤더를 접으면(▲ 버튼) 핵심 정보만 표시되어 그리드 영역을 더 넓게 활용할 수 있습니다. 다시 펼치려면 ▼ 버튼을 누르세요.

## 환자 정보 확인

차트 헤더에서 **환자 이름**을 클릭하면 환자 상세 정보를 확인할 수 있습니다.

표시되는 정보:
- 환자명
- 종 / 품종 / 나이 / 성별
- 입원 일차 (연속 차트가 있는 경우)

[스크린샷: 차트 헤더 — 환자명 클릭 시 환자 정보 패널]

---

## 차트 날짜 이동

차트 상세 화면의 **날짜 네비게이터**를 사용하여 같은 환자의 이전/이후 차트로 이동할 수 있습니다.

[스크린샷: 차트 상세 화면 내 날짜 네비게이터 — 좌우 화살표]

- **←**: 이전 날짜 차트로 이동
- **→**: 다음 날짜 차트로 이동
- 해당 날짜에 차트가 없으면 새 차트 생성 여부를 묻습니다.

---

## 차트 삭제

1. 차트 상세 화면에서 드롭다운 메뉴(⋯)를 엽니다.
2. **"차트 삭제"**를 선택합니다.
3. 확인 다이얼로그에서 **"삭제"**를 누르면 차트가 영구 삭제됩니다.

> **주의:** 삭제된 차트는 복구할 수 없습니다. 차트에 포함된 모든 셀 데이터, 처치행, 수액행이 함께 삭제됩니다.
