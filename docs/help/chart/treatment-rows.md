# 처치행 관리

처치행은 약물 투여, 시술, 처치 등을 기록하기 위한 **동적 행**입니다. 필요에 따라 행을 추가, 삭제, 순서 변경할 수 있습니다.

---

## 처치행이란?

차트에서 고정행(Vital) 아래에 위치하며, 각 행이 하나의 처치 항목을 나타냅니다.

![처치행 그리드](/images/screenshots/help/chart-grid-treatment-rows.png)

- 행 왼쪽에 **처치 항목명**이 표시됩니다. (예: Cefazolin, Meloxicam)
- 각 시간대 셀에 투여 여부나 투여량을 기록합니다.

---

## 처치행 추가

차트에 처치행을 추가하는 방법은 두 가지입니다.

### 방법 1: 빈 행에 이름 입력
차트에는 기본적으로 빈 처치행이 표시됩니다. 빈 행의 이름 영역을 클릭하고 **처치 항목명을 입력**하면 자동으로 행이 생성됩니다.

![처치행 입력](/images/screenshots/help/treatment-row-input.png)

### 방법 2: 행 삽입
기존 행의 오른쪽에 있는 **"+" 버튼**을 클릭하면 해당 행 아래에 새 행이 삽입됩니다. **"×" 버튼**으로 행을 삭제할 수 있습니다.

![처치행 추가/삭제 버튼](/images/screenshots/help/treatment-row-context-menu.png)

---

## 항목명 입력 & 제품 연결

### 직접 입력
처치행 이름 영역에 약물이나 시술 이름을 직접 타이핑합니다. 이전에 사용한 항목명이 **자동완성 제안**으로 표시됩니다.

![자동완성 드롭다운](/images/screenshots/help/treatment-row-autocomplete.png)

### 제품(약물) 연결
등록된 약물 제품을 선택하면 **용량 자동 계산** 기능이 활성화됩니다.

1. 항목명 입력 시 등록된 제품 목록에서 선택합니다.
2. 제품이 연결되면 행 왼쪽에 **파란색 세로선**이 표시됩니다.
3. 환자 체중 기반으로 투여 용량이 자동 계산되어 뱃지로 표시됩니다.

![제품 연결된 처치행](/images/screenshots/help/treatment-row-product-linked.png)

> **팁:** 제품 연결을 해제하려면 행 우측 상단의 **X** 버튼을 클릭하세요.

---

## 행 순서 변경

처치행의 순서를 변경하는 방법은 두 가지입니다.

### 방법 1: 드래그 앤 드롭
행 왼쪽의 **드래그 핸들(⠿)** 을 잡고 위아래로 끌어 원하는 위치에 놓습니다.

### 방법 2: 이동 버튼
행의 드래그 핸들(⠿) 옆 **이동 버튼**을 사용하여 위아래로 이동합니다.

![처치행 드래그 앤 드롭](/images/screenshots/help/treatment-row-drag.png)

---

## 행 삭제

1. 삭제하려는 행의 오른쪽에 있는 **"×" 버튼**을 클릭합니다.

> **주의:** 행을 삭제하면 해당 행의 모든 셀 데이터(시간대별 입력값, 마킹 등)도 함께 삭제됩니다. 삭제는 되돌릴 수 없습니다.

---

## 용량 자동 계산

제품이 연결된 처치행에서는 **환자 체중 기반의 용량 자동 계산**이 제공됩니다.

- 처치행 이름 옆에 계산된 투여량이 **뱃지**로 표시됩니다.
- 예) `0.25 ml` — 체중 5kg 환자에게 제품 농도 기준으로 계산된 투여량

### 약물 계산기
차트 헤더의 **"약물"** 버튼을 누르면 약물 계산기 패널이 열립니다. 여기서 더 상세한 용량 계산을 수행할 수 있습니다.

![약물 계산기 패널](/images/screenshots/help/drug-calculator-panel.png)

> **팁:** 차트의 체중 셀에 새 값을 입력하면, 모든 연결된 처치행의 용량이 자동으로 재계산됩니다.
