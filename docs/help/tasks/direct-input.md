# 직접 입력

직접 입력은 업무 목록에서 **차트를 열지 않고** 바로 값을 입력하고 업무를 완료하는 기능입니다. 바쁜 진료 현장에서 빠르게 업무를 처리할 때 유용합니다.

---

## 직접 입력이란?

업무 목록의 각 업무 카드에서 **"직접 입력"** 버튼을 누르면 입력 모달이 열립니다. 모달에서 값을 입력하면 해당 차트의 셀에 자동 저장되고, 업무가 완료 처리됩니다.

![직접 입력 버튼](/images/screenshots/help/task-direct-input-btn.png)

---

## 사용 방법

### Step 1: 업무 카드에서 직접 입력 열기

업무 목록 페이지에서 입력하려는 업무 카드의 **"직접 입력"** 버튼을 탭합니다.

### Step 2: 값 입력

입력 모달이 열리면 아래 정보가 표시됩니다.

- **환자 이름**: 어떤 환자인지
- **항목명**: 어떤 처치/측정인지
- **시간대**: 몇 시 업무인지
- **용량 정보**: 제품 연결 시 투여 용량 참고

![직접 입력 모달](/images/screenshots/help/task-direct-input-modal.png)

입력 필드에 값을 입력합니다. 예) 체온: `38.5`, Cefazolin: `0.25ml`

> **팁:** 빠른 입력이 지원되는 항목(구토, 배변, 배뇨, 식이)에서는 선택 버튼이 함께 표시됩니다. 한 번 탭으로 빠르게 입력할 수 있습니다.

### Step 3: 완료

**"완료"** 버튼을 누르면:
1. 입력한 값이 차트의 해당 셀에 **자동 저장**됩니다.
2. 해당 셀이 **완료 상태**로 변경됩니다.
3. 모달이 닫히고, 업무 카드에 완료 애니메이션이 표시됩니다.

![완료 페이드 아웃](/images/screenshots/help/task-complete-fadeout.png)

---

## 빠른 완료 (값 입력 없이)

값을 입력할 필요 없이 단순히 **"했다"** 는 기록만 남기면 되는 업무도 있습니다.

업무 카드의 **"빠른 완료"** 버튼을 누르면 값 입력 없이 바로 완료 처리됩니다.

> **팁:** 빠른 완료는 별도 모달 없이 즉시 처리되어, 반복 업무를 빠르게 소화할 때 편리합니다.

---

## 차트로 이동

더 상세한 확인이나 편집이 필요한 경우, 직접 입력 모달이나 업무 카드에서 **"차트로 이동"** 버튼을 누르면 해당 차트의 해당 셀 위치로 바로 이동합니다.

[스크린샷: 직접 입력 모달 하단 "차트로 이동" 링크]

> **팁:** 차트로 이동하면 해당 셀이 **하이라이트**되어 위치를 쉽게 찾을 수 있습니다.
