# 환자 등록 & 관리

VetSync에서 환자(동물) 정보를 등록, 검색, 수정하는 방법을 안내합니다.

## 환자 목록 접근

- **PC**: 상단 네비게이션에서 **"환자/보호자"** 탭을 클릭합니다.
- **모바일**: 하단 네비게이션 **"더보기"** 탭 → **"환자/보호자"** 메뉴를 탭합니다.

환자/보호자 페이지에 들어가면 상단에 **"환자 관리"** / **"보호자 관리"** 탭이 있습니다. "환자 관리" 탭이 기본 선택되어 있습니다.

![환자 탭 UI](/images/screenshots/help/patient-tab-ui.png)

## 새 환자 등록

1. 우측 상단(PC) 또는 헤더(모바일)의 **"+ 환자 등록"** 버튼을 탭합니다.
2. 환자 등록 모달이 표시됩니다.
3. 아래 정보를 입력합니다:

| 필드 | 필수 | 설명 |
|------|------|------|
| 이름 | O | 환자(동물)의 이름 |
| 종 | O | 개, 고양이, 토끼 등 |
| 품종 | | 포메라니안, 러시안블루 등 |
| 나이 | | 숫자(0~100) |
| 성별 | | 수컷 / 암컷 / 중성화수컷(CM) / 중성화암컷(SF) / 미상 |
| CPCR/DNR | | CPCR/DNR 여부 선택 |
| 체중 (kg) | | 숫자, 소수점 2자리까지 (0.01~999.99) |
| 카프번호 | | 혈압 측정 시 사용하는 커프 번호 (예: #1) |
| 메모 | | 특이사항 기록 |
| 보호자 | | 기존 보호자 검색 및 연결 |

4. **"저장"** 버튼을 탭하여 등록을 완료합니다.

![환자 등록 모달](/images/screenshots/help/patient-register-modal.png)

> **팁:** 체중은 약물 용량 계산기에서 자동으로 사용됩니다. 정확한 체중을 입력해주세요.

## 환자 검색

환자 목록 상단의 **검색란**에 환자 이름을 입력하면 실시간으로 필터링됩니다.

![환자 검색](/images/screenshots/help/patient-search.png)

## 환자 상세 보기

1. 환자 목록에서 원하는 환자 카드를 클릭/탭합니다.
2. 환자 상세 모달이 열리며, 다음 정보가 표시됩니다:
   - 기본 정보: 이름, 종, 품종, 나이, 성별, 체중, CPCR 여부
   - 카프번호
   - 메모
   - 연결된 보호자 정보 (이름, 전화번호)

![환자 상세 모달](/images/screenshots/help/patient-detail-modal.png)

> **팁:** 연결된 보호자 이름을 클릭하면 해당 보호자의 상세 정보로 바로 이동합니다.

## 환자 정보 수정

1. 환자 상세 모달에서 **"수정"** 버튼을 탭합니다.
2. 환자 수정 모달이 열립니다.
3. 수정할 항목을 변경합니다.
4. 수정 모드에서는 추가로 **상태** 필드가 표시됩니다:
   - **활성**: 현재 진료 중인 환자 (기본값)
   - **비활성**: 퇴원/전원 등으로 현재 진료하지 않는 환자
5. **"저장"** 버튼을 탭하여 수정을 완료합니다.

![환자 수정 모달](/images/screenshots/help/patient-edit-modal.png)

## 체중 업데이트

환자의 체중이 변경된 경우:

1. 환자 상세 → 수정 모달에서 **체중 (kg)** 필드를 업데이트합니다.
2. 저장하면 이후 생성되는 차트에서 업데이트된 체중이 반영됩니다.
3. 약물 용량 계산기에서도 새 체중이 자동으로 사용됩니다.

> **주의:** 체중 변경은 기존 차트의 계산 결과에 소급 적용되지 않습니다. 이미 계산된 차트의 용량을 업데이트하려면 해당 차트의 계산기에서 다시 적용해주세요.

## 카프번호 설정

혈압 측정 시 사용하는 커프 번호를 환자별로 관리할 수 있습니다:

1. 환자 등록 또는 수정 모달에서 **"카프번호"** 필드에 입력합니다.
2. 예: `#1`, `#2`, `소형` 등 자유 형식으로 입력 가능합니다.
3. 차트에서 혈압 측정 시 어떤 커프를 사용해야 하는지 참고할 수 있습니다.
