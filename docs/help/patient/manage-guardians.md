# 보호자 등록 & 관리

VetSync에서 보호자(반려인) 정보를 등록, 검색, 수정하는 방법을 안내합니다.

## 보호자 목록 접근

1. **PC**: 상단 네비게이션에서 **"환자/보호자"** 탭을 클릭합니다.
2. **모바일**: 하단 네비게이션 **"더보기"** 탭 → **"환자/보호자"** 메뉴를 탭합니다.
3. 환자/보호자 페이지에서 상단의 **"보호자 관리"** 탭을 선택합니다.

![보호자 탭](/images/screenshots/help/guardian-tab.png)

## 새 보호자 등록

1. 우측 상단(PC) 또는 헤더(모바일)의 **"+ 보호자 등록"** 버튼을 탭합니다.
2. 보호자 등록 모달이 표시됩니다.
3. 아래 정보를 입력합니다:

| 필드 | 필수 | 설명 |
|------|------|------|
| 이름 | O | 보호자 이름 |
| 전화번호 | O | 연락처 (예: 010-1234-5678) |
| 주소 | | 보호자 주소 |
| 이메일 | | 이메일 주소 |
| 메모 | | 특이사항 기록 |

4. **"저장"** 버튼을 탭하여 등록을 완료합니다.

![보호자 등록 모달](/images/screenshots/help/guardian-register-modal.png)

> **주의:** 같은 병원 내에서 동일한 전화번호로 중복 등록할 수 없습니다. 이미 등록된 번호라면 "이미 등록된 전화번호입니다" 메시지가 표시됩니다.

> **팁:** 전화번호는 하이픈(-) 유무와 관계없이 입력 가능합니다. 시스템이 자동으로 정규화합니다.

## 보호자 검색

보호자 목록 상단의 **검색란**에 보호자 이름 또는 전화번호를 입력하면 실시간으로 필터링됩니다.

![보호자 검색](/images/screenshots/help/guardian-search.png)

## 보호자 상세 보기

1. 보호자 목록에서 원하는 보호자 카드를 클릭/탭합니다.
2. 보호자 상세 모달이 열리며, 다음 정보가 표시됩니다:
   - 이름, 전화번호, 주소, 이메일
   - 메모
   - 연결된 환자 목록

![보호자 상세 모달](/images/screenshots/help/guardian-detail-modal.png)

> **팁:** 보호자 상세에서 연결된 환자 이름을 클릭하면 환자 관리 탭으로 전환되어 해당 환자를 바로 찾을 수 있습니다.

## 보호자 정보 수정

1. 보호자 상세 모달에서 **"수정"** 버튼을 탭합니다.
2. 보호자 수정 모달이 열립니다.
3. 수정할 항목을 변경합니다.
4. **"저장"** 버튼을 탭하여 수정을 완료합니다.

![보호자 수정 모달](/images/screenshots/help/guardian-edit-modal.png)

> **주의:** 전화번호를 다른 보호자가 이미 사용 중인 번호로 변경하면 "이미 등록된 전화번호입니다" 오류가 발생합니다.
