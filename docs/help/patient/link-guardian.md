# 환자-보호자 연결

환자와 보호자를 연결하여 진료 기록에서 보호자 정보를 함께 관리할 수 있습니다.

## 환자 등록 시 보호자 연결

새 환자를 등록할 때 보호자를 바로 연결할 수 있습니다.

1. **"+ 환자 등록"** 버튼을 탭합니다.
2. 환자 정보를 입력합니다.
3. 폼 하단의 **"보호자"** 항목에서 보호자를 검색합니다.
4. 기존 보호자 목록에서 선택합니다.
5. **"저장"** 버튼을 탭하면 환자 등록과 보호자 연결이 동시에 완료됩니다.

![보호자 선택](/images/screenshots/help/patient-guardian-select.png)

> **팁:** 보호자가 아직 등록되지 않은 경우, 먼저 보호자 관리 탭에서 보호자를 등록한 후 환자와 연결해주세요.

## 환자 수정에서 보호자 변경

이미 등록된 환자의 보호자를 변경하거나 새로 연결할 수 있습니다.

1. 환자 목록에서 대상 환자를 선택하여 상세 모달을 엽니다.
2. **"수정"** 버튼을 탭합니다.
3. **"보호자"** 항목에서:
   - 다른 보호자를 검색하여 선택하면 보호자가 **변경**됩니다.
   - 보호자 선택을 해제하면 보호자 연결이 **해제**됩니다.
4. **"저장"** 버튼을 탭하여 변경을 완료합니다.

![보호자 변경](/images/screenshots/help/patient-edit-guardian-change.png)

## 한 보호자에 여러 환자 연결

한 명의 보호자에게 여러 마리의 환자를 연결할 수 있습니다.

- 각 환자의 등록/수정 시 동일한 보호자를 선택하면 됩니다.
- 보호자 상세 모달에서 연결된 모든 환자 목록을 확인할 수 있습니다.

예시: 보호자 "홍길동"에게 "초코"(개), "나비"(고양이) 두 마리가 연결된 경우

![다중 환자 연결](/images/screenshots/help/guardian-multi-patients.png)

## 환자-보호자 간 빠른 이동

환자/보호자 페이지에서 연결된 정보 사이를 빠르게 이동할 수 있습니다:

- **환자 상세 → 보호자**: 환자 상세 모달에서 보호자 이름을 클릭하면 해당 보호자의 상세 정보가 열립니다.
- **보호자 상세 → 환자**: 보호자 상세 모달에서 연결된 환자 이름을 클릭하면 환자 관리 탭으로 전환되어 해당 환자를 찾을 수 있습니다.

![환자-보호자 연결](/images/screenshots/help/patient-to-guardian-link.png)

> **팁:** 이 기능을 활용하면 보호자의 다른 반려동물 정보를 빠르게 확인할 수 있습니다.
