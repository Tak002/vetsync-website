# 약물/수액 제품 등록

병원에서 사용하는 약물 및 수액 제품을 등록하고 관리하는 방법을 안내합니다. 등록된 제품은 약물 계산기와 차트 처치행에서 활용됩니다.

> **참고:** 제품 관리는 **마스터(MASTER)** 권한이 필요합니다. 일반 직원은 **설정 > 참고 정보** 탭에서 등록된 제품 목록을 읽기 전용으로 확인할 수 있습니다.

---

## 접근 방법

1. **설정** > **약물·수액** 탭으로 이동합니다
2. "약물 제품" 섹션을 확인합니다

[스크린샷: 약물 제품 관리 섹션 — 테이블 형식의 제품 목록]

---

## 제품 목록

등록된 제품은 테이블 형식으로 표시됩니다:

| 열 | 설명 |
|----|------|
| **제품명** | 제품 이름 (예: "세파졸린 주사액") |
| **약물명** | 연결된 약물 성분 (미연결 시 "-") |
| **농도** | 농도값 + 단위 (예: "100mg/ml") |
| **가격** | 단가 (미입력 시 "-") |

---

## 새 제품 등록

1. **+ 제품 추가** 버튼을 클릭합니다
2. 테이블 위에 입력 폼이 나타납니다

[스크린샷: 제품 추가 폼 — 약물 검색, 제품명, 농도, 단위, 가격 입력]

### 입력 항목

#### 약물 연결 (선택)

글로벌 약물 데이터베이스(Plumb's 10th Edition 기반, 726종)에서 성분을 선택합니다.

1. 검색창에 약물명을 입력합니다 (예: "cefa")
2. 검색 결과에서 해당 약물을 클릭합니다
   - 각 결과에 **약물명**과 **카테고리**(SYSTEMIC/OPHTHALMIC/DERMATOLOGIC)가 표시됩니다
   - 최대 50개까지 표시되며, 초과 시 "...외 N건 (검색어를 입력하세요)" 안내가 나옵니다
3. 선택된 약물이 표시되며, **X 버튼**으로 해제할 수 있습니다

> **팁:** 약물 연결은 선택사항입니다. 약물 성분이 목록에 없는 제품이나 소모품도 약물 미선택 상태로 등록할 수 있습니다.

#### 제품명 (필수)

- 병원에서 실제 사용하는 제품 이름을 입력합니다
- 최대 200자
- **같은 병원 내에서 제품명이 중복되면 등록할 수 없습니다**

#### 농도 설정 (필수)

약물의 농도를 3개 필드로 입력합니다:

| 필드 | 설명 | 예시 |
|------|------|------|
| **농도값** | 숫자 (양수, 소수점 가능) | `100` |
| **농도 단위** | 드롭다운: mg / IU / mcg | `mg` |
| **단위** | 드롭다운: ml / 정 / 캡슐 / g / vial / patch / tube / drop | `ml` |

조합 예시:
- 세파졸린 주사액: `100` `mg` / `ml` → **100mg/ml**
- 프레드니솔론 정: `5` `mg` / `정` → **5mg/정**
- 인슐린 주사액: `100` `IU` / `ml` → **100IU/ml**
- 피프로닐 스팟온: `100` `mg` / `tube` → **100mg/tube**

#### 가격 (선택)

- 단가를 원(원) 단위로 입력합니다
- 미입력 시 "-"로 표시됩니다

### 저장

3. **저장 아이콘** (체크)을 클릭합니다
4. 제품이 목록에 추가됩니다

---

## 제품 수정

1. 수정할 제품 행의 **편집 아이콘** (연필)을 클릭합니다
2. 테이블 행이 입력 폼으로 전환됩니다
3. 필요한 항목을 수정합니다
4. **저장 아이콘** (체크)을 클릭하거나, **취소 아이콘** (X)으로 되돌립니다

---

## 제품 삭제

1. 삭제할 제품 행의 **삭제 아이콘** (휴지통)을 클릭합니다
2. 확인 다이얼로그:
   > "[제품명]" 제품을 삭제하시겠습니까?
3. **삭제**를 클릭하면 제품이 제거됩니다

> **주의:** 제품을 삭제해도 해당 제품이 사용된 기존 차트 기록은 유지됩니다.

---

## 약물 데이터베이스 참고

VetSync에는 **Plumb's Veterinary Drug Handbook 10th Edition** 기반의 글로벌 약물 데이터베이스가 내장되어 있습니다.

- **총 726종**의 약물 성분 정보
- 카테고리: SYSTEMIC (587종) / OPHTHALMIC (86종) / DERMATOLOGIC (53종)
- 이 데이터베이스는 읽기 전용이며, 제품 등록 시 약물 연결에 활용됩니다
