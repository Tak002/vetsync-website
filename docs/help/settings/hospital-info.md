# 병원 정보 & 차트 잠금 설정

병원 기본 정보를 수정하고, 차트 편집 잠금을 설정하는 방법을 안내합니다.

---

## 병원 정보 수정

### 접근 방법

1. **설정** > **병원 운영** 탭으로 이동합니다
2. "병원 정보" 섹션을 확인합니다

> **참고:** 병원 정보 수정은 **마스터(MASTER)** 권한이 필요합니다. 일반 직원은 정보를 조회만 할 수 있으며, "병원 정보 수정은 관리자 권한이 필요합니다" 메시지가 표시됩니다.

[스크린샷: 병원 정보 폼 — 병원명, 주소, 전화번호 입력 필드]

### 수정 가능한 항목

| 항목 | 필수 여부 | 설명 |
|------|-----------|------|
| **병원명** | 필수 | 병원 이름 |
| **주소** | 선택 | 병원 주소 |
| **전화번호** | 선택 | 병원 대표 전화번호 |

### 수정 방법

1. 각 필드를 클릭하여 내용을 수정합니다
2. **저장** 버튼을 클릭합니다
3. 변경된 정보가 즉시 반영됩니다

---

## 차트 편집 잠금

특정 기간이 지난 차트의 수정을 방지하는 기능입니다. 진료 기록의 무결성을 보호할 때 유용합니다.

### 접근 방법

1. **설정** > **차트 설정** 탭으로 이동합니다
2. "차트 수정 잠금" 섹션을 확인합니다

> **참고:** 차트 편집 잠금 설정은 **마스터(MASTER)** 권한이 필요합니다.

[스크린샷: 차트 수정 잠금 설정 — 토글 스위치 + 일수 입력]

### 설정 방법

1. **"차트 수정 잠금 사용"** 토글을 **ON**으로 전환합니다
2. 잠금까지의 **일수**를 입력합니다 (1~365일)
   - 예: `3`을 입력하면, 차트 작성일로부터 3일 후 수정이 잠깁니다
3. **저장** 버튼을 클릭합니다

### 잠금 동작

- **활성화 시**: "차트 작성일로부터 N일 이후 수정이 잠깁니다" 안내가 표시됩니다
- **비활성화 시**: "비활성화 시 날짜에 관계없이 항상 수정 가능합니다" 안내가 표시됩니다
- 잠긴 차트는 셀 입력, 행 추가/삭제, 헤더 편집이 모두 차단됩니다

> **팁:** 입력값이 비어 있으면 기본값 **1일**로 설정됩니다. 저장 버튼은 설정이 변경된 경우에만 활성화됩니다.

> **주의:** 잠금 설정은 병원 전체에 적용됩니다. 설정 변경 후에는 모든 직원의 차트 편집에 영향을 미칩니다.
