# 스태프 관리

병원에 소속된 직원을 초대하고, 권한을 설정하며, 가입 요청을 관리하는 방법을 안내합니다.

> **참고:** 스태프 관리 기능은 **마스터(MASTER)** 권한을 가진 관리자만 사용할 수 있습니다.

![병원 운영 탭](/images/screenshots/help/settings-hospital-tab.png)

---

## 권한 체계

VetSync는 4단계 권한 체계를 사용합니다. **상위 권한은 하위 권한의 모든 기능을 포함**합니다.

| 권한 | 이름 | 주요 기능 |
|------|------|-----------|
| **MASTER** | 마스터 | 모든 기능 + 병원 설정 + 스태프 관리 + 진료 템플릿 관리 |
| **PLANNER** | 기획 | 차트 작성 + 업무 관리 + 환자/보호자 관리 + 게시판 |
| **EXECUTOR** | 실행 | 차트 입력 + 업무 완료 처리 + 환자 조회 |
| **RECEPTIONIST** | 접수 | 기본 조회 + 접수 업무 |

![권한 설명](/images/screenshots/help/settings-role-popover.png)

> **팁:** 권한 옆의 **?** 아이콘을 클릭하면 각 권한에 대한 자세한 설명을 볼 수 있습니다. 현재 내 권한은 강조 표시됩니다.

---

## 초대 코드로 직원 초대하기

새 직원이 병원에 합류하려면 **초대 코드**가 필요합니다.

### 초대 코드 생성

1. **설정** > **병원 운영** 탭으로 이동합니다
2. "초대 코드" 섹션에서 **초대 코드 생성** 버튼을 클릭합니다
3. 6자리 코드가 생성되어 화면에 표시됩니다

![초대 코드](/images/screenshots/help/settings-invite-code.png)

### 초대 코드 공유

- **복사** 버튼을 클릭하면 코드가 클립보드에 복사됩니다
- 복사한 코드를 메신저, 문자 등으로 직원에게 전달하세요

### 초대 코드 재생성

- **재생성** 버튼을 클릭하면 새로운 코드가 발급됩니다

> **주의:** 재생성하면 **기존 코드는 즉시 비활성화**됩니다. 이미 코드를 공유한 직원이 아직 가입하지 않았다면, 새 코드를 다시 전달해야 합니다.

## 가입 요청 승인/거절

직원이 초대 코드를 입력하고 가입 신청을 하면, 관리자의 승인이 필요합니다.

### 가입 요청 확인

1. **설정** > **병원 운영** 탭으로 이동합니다
2. "가입 요청" 섹션에서 대기 중인 요청을 확인합니다
3. 각 요청에는 **직원명**과 **이메일**이 표시됩니다

![가입 요청 목록](/images/screenshots/help/settings-join-requests.png)

### 권한 선택 후 승인

1. 요청자 카드의 **권한 드롭다운**에서 적절한 권한을 선택합니다
   - 기본값: RECEPTIONIST (접수)
   - 선택 가능: RECEPTIONIST / EXECUTOR / PLANNER
2. **승인 버튼** (체크 아이콘)을 클릭합니다
3. "가입 요청을 승인했습니다" 메시지가 표시됩니다

> **팁:** 승인 시 선택한 권한으로 즉시 계정이 활성화됩니다. 권한은 나중에 직원 상세에서 변경할 수 있습니다.

### 가입 요청 거절

1. **거절 버튼** (X 아이콘)을 클릭합니다
2. "가입 요청을 거절했습니다" 메시지가 표시됩니다

> **참고:** 대기 중인 가입 요청이 없으면 "대기 중인 가입 요청이 없습니다"라고 표시됩니다.

## 직원 목록 조회

### 목록 보기

1. **설정** > **병원 운영** 탭 > "직원 관리" 섹션으로 이동합니다
2. 병원에 소속된 모든 직원이 목록으로 표시됩니다

![직원 목록](/images/screenshots/help/settings-staff-list.png)

### 표시 정보

각 직원 행에는 다음 정보가 표시됩니다:

| 항목 | 설명 |
|------|------|
| **이름** | 직원 이름 (굵은 글씨) |
| **이메일** | 로그인 이메일 (회색) |
| **직책** | 수의사, 테크니션 등 (회색) |
| **권한 배지** | MASTER(파랑) / PLANNER(시안) / EXECUTOR(민트) / RECEPTIONIST(회색) |
| **상태 배지** | 활성(민트) / 비활성(회색) |

### 직원 검색

- 목록 상단의 검색창에 **이름** 또는 **직책**을 입력하면 실시간 필터링됩니다

## 직원 상세 정보 & 관리

직원 행을 클릭하면 **상세 모달**이 열립니다.

![직원 상세](/images/screenshots/help/settings-staff-detail.png)

### 표시 항목

| 항목 | 설명 | 수정 가능 |
|------|------|-----------|
| 이름 | 직원 이름 | 수정 버튼으로 이동 |
| 이메일 | 로그인 이메일 | 불가 (읽기 전용) |
| 직책 | 수의사, 테크니션 등 | 수정 버튼으로 이동 |
| 부서 | 외과, 내과 등 | 수정 버튼으로 이동 |
| 전화번호 | 연락처 | 수정 버튼으로 이동 |
| 권한 | MASTER~RECEPTIONIST | 드롭다운으로 즉시 변경 |
| 상태 | 활성/비활성 | 버튼으로 전환 |

### 권한 변경

1. 상세 모달에서 **권한 드롭다운**을 클릭합니다
2. 변경할 권한을 선택합니다 (MASTER / PLANNER / EXECUTOR / RECEPTIONIST)
3. 즉시 반영되며 "권한이 [역할명](으)로 변경되었습니다" 메시지가 표시됩니다

> **주의:** MASTER 권한은 병원 설정과 직원 관리가 가능한 최고 권한입니다. 신중하게 부여하세요.

### 직원 정보 수정

1. 상세 모달 우측 상단의 **수정 아이콘** (연필)을 클릭합니다
2. **직원 편집 폼**이 열립니다
3. 수정 가능한 필드:
   - **이름** (필수, 1자 이상)
   - **직책** (선택, 예: 수의사, 테크니션)
   - **부서** (선택, 예: 외과, 내과)
   - **전화번호** (선택, 예: 010-1234-5678)
4. **저장** 버튼을 클릭합니다
5. "직원 정보가 수정되었습니다" 메시지가 표시됩니다

![직원 편집](/images/screenshots/help/settings-staff-edit.png)

### 계정 비활성화/활성화

비활성화된 계정은 **로그인이 불가능**합니다. 퇴사한 직원이나 임시로 접근을 차단할 때 사용합니다.

**비활성화하기:**
1. 상세 모달에서 **계정 비활성화** 버튼을 클릭합니다
2. 확인 다이얼로그: "계정을 비활성화하시겠습니까? 비활성화된 계정은 더 이상 로그인할 수 없습니다."
3. **비활성화**를 클릭하면 즉시 적용됩니다

**다시 활성화하기:**
1. 비활성화된 직원의 상세 모달에서 **계정 활성화** 버튼을 클릭합니다
2. 즉시 로그인이 다시 가능해집니다

> **팁:** 비활성화해도 직원의 기존 차트 기록은 삭제되지 않습니다. 필요 시 언제든 다시 활성화할 수 있습니다.
