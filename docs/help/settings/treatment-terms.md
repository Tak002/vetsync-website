# 예시어, 투여 빈도, 수액 기본액 관리

차트 작성에 사용되는 보조 항목들을 설정하는 방법을 안내합니다. 예시어(처치 항목명), 투여 빈도, 수액 기본액을 병원에 맞게 커스터마이징할 수 있습니다.

> **참고:** 이 설정들은 **마스터(MASTER)** 권한이 필요합니다. 일반 직원은 **설정 > 참고 정보** 탭에서 읽기 전용으로 확인할 수 있습니다.

---

## 예시어 관리

예시어는 처치행에 입력할 수 있는 **사전 정의된 처치 항목 이름**입니다. 자주 사용하는 처치명을 미리 등록해두면 차트 작성 시 빠르게 선택할 수 있습니다.

### 접근 방법

1. **설정** > **차트 설정** 탭으로 이동합니다
2. "예시어" 섹션을 확인합니다

![예시어 관리](/images/screenshots/help/settings-treatment-terms.png)

### 예시어 추가

1. **+ 예시어 추가** 버튼을 클릭합니다
2. 인라인 편집 폼이 나타납니다
3. **항목명**을 입력합니다 (필수, 최대 100자)
   - 예: "항생제 투여", "수액 교체", "드레싱", "산소 공급"
4. **저장 아이콘** (체크)을 클릭합니다

> **참고:** 예시어는 **최대 50개**까지 등록할 수 있습니다. 한도에 도달하면 추가 버튼이 비활성화됩니다.

### 예시어 수정

1. 수정할 항목의 **편집 아이콘** (연필)을 클릭합니다
2. 항목명을 수정합니다
3. **저장 아이콘** (체크)을 클릭합니다

### 예시어 삭제

1. 삭제할 항목의 **삭제 아이콘** (휴지통)을 클릭합니다
2. 확인 다이얼로그:
   > "[항목명]" 예시어를 삭제하시겠습니까?
3. **삭제**를 클릭합니다

## 투여 빈도 설정

투여 빈도는 약물의 **투여 횟수와 시간**을 정의합니다. SID(1일 1회), BID(1일 2회) 등의 코드를 등록하고, 각 코드에 투여 시간을 지정합니다.

### 접근 방법

1. **설정** > **약물·수액** 탭으로 이동합니다
2. "투여 횟수" 섹션을 확인합니다

![투여 횟수 관리](/images/screenshots/help/settings-frequency.png)

### 표시 정보

각 투여 빈도 항목에는 다음이 표시됩니다:

| 항목 | 설명 | 예시 |
|------|------|------|
| **코드** | 약어 (굵은 글씨) | SID, BID, TID, QID |
| **설명** | 한국어 설명 | 1일 1회, 1일 2회 |
| **시간** | 투여 시간 목록 | 8시, 12시, 18시 |

### 투여 빈도 추가

1. **+ 투여 횟수 추가** 버튼을 클릭합니다
2. 인라인 편집 폼이 나타납니다
3. 입력 필드:

| 필드 | 규칙 | 예시 |
|------|------|------|
| **코드** (필수) | A-Z, a-z, 0-9, -, _ 만 허용. 최대 20자. 자동 대문자 변환 | `BID` |
| **설명** (필수) | 최대 50자 | `1일 2회` |
| **투여 시간** (필수) | 0~23시 중 최소 1개 선택 | `8시, 20시` |

4. **투여 시간 선택**: 0~23시 버튼이 8x3 격자로 표시됩니다
   - 클릭하면 **금색**으로 활성화됩니다
   - 다시 클릭하면 비활성화됩니다
   - 최소 1개 이상 선택해야 합니다

![투여 빈도 편집](/images/screenshots/help/settings-frequency-edit.png)

5. **저장 아이콘** (체크)을 클릭합니다

> **참고:** 투여 빈도는 **최대 20개**까지 등록할 수 있습니다.

> **주의:** 동일한 코드가 이미 존재하면 "이미 동일한 코드의 투여 횟수가 존재합니다" 오류가 표시됩니다.

### 투여 빈도 수정

1. 수정할 항목의 **편집 아이콘** (연필)을 클릭합니다
2. 코드, 설명, 투여 시간을 수정합니다
3. **저장 아이콘** (체크)을 클릭합니다

### 투여 빈도 삭제

1. 삭제할 항목의 **삭제 아이콘** (휴지통)을 클릭합니다
2. 확인 다이얼로그:
   > "[코드] ([설명])"을(를) 삭제하시겠습니까? 기존 프리셋의 시간 설정에는 영향 없습니다.
3. **삭제**를 클릭합니다

> **참고:** 투여 빈도를 삭제해도 이미 프리셋에 설정된 시간 정보는 유지됩니다.

### 자주 사용하는 투여 빈도 예시

| 코드 | 설명 | 투여 시간 |
|------|------|-----------|
| SID | 1일 1회 | 8시 |
| BID | 1일 2회 | 8시, 20시 |
| TID | 1일 3회 | 8시, 14시, 20시 |
| QID | 1일 4회 | 6시, 12시, 18시, 24시 |
| EOD | 격일 | 8시 |
| PRN | 필요시 | (시간 자유 선택) |

## 수액 기본액 관리

수액 기본액은 수액행에서 선택할 수 있는 **기본 수액 종류**를 정의합니다.

### 접근 방법

1. **설정** > **약물·수액** 탭으로 이동합니다
2. "수액 베이스" 섹션을 확인합니다

![수액 베이스 관리](/images/screenshots/help/settings-fluid-bases.png)

### 표시 정보

| 열 | 설명 |
|----|------|
| **베이스명** | 수액 이름 (예: "생리식염수") |
| **설명** | 부가 설명 (선택, 예: "유지액으로 사용") |

### 수액 기본액 추가

1. **+ 수액 베이스 추가** 버튼을 클릭합니다
2. 테이블 위에 입력 폼이 나타납니다
3. 입력 필드:
   - **베이스명** (필수, 최대 100자): 예) "0.9% NaCl (생리식염수)"
   - **설명** (선택, 최대 500자): 예) "등장액, 일반 유지 수액"
4. **저장 아이콘** (체크)을 클릭합니다

### 수액 기본액 수정

1. 수정할 항목의 **편집 아이콘** (연필)을 클릭합니다
2. 테이블 행이 입력 폼으로 전환됩니다
3. 내용을 수정하고 **저장 아이콘** (체크)을 클릭합니다

### 수액 기본액 삭제

1. 삭제할 항목의 **삭제 아이콘** (휴지통)을 클릭합니다
2. 확인 다이얼로그:
   > "[베이스명]"을(를) 삭제하시겠습니까?
3. **삭제**를 클릭합니다

### 자주 등록하는 수액 기본액 예시

| 베이스명 | 설명 |
|----------|------|
| 0.9% NaCl (생리식염수) | 등장액, 가장 범용적 |
| Lactated Ringer's (LRS) | 등장액, 전해질 보충 |
| 5% Dextrose (D5W) | 저장액, 수분/칼로리 보충 |
| 0.45% NaCl (Half-NS) | 저장액, 유지 수액 |
| Hartmann's Solution | LRS와 유사, 전해질 보충 |
