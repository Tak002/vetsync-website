# 댓글 & 익명 기능

게시판 글에 댓글을 달고, 익명 기능을 활용하여 자유롭게 의견을 나눌 수 있습니다.

## 댓글 작성

1. 글 상세 페이지 하단의 **댓글 입력란**에 내용을 입력합니다.
2. 필요하면 **익명 옵션**을 선택합니다.
3. **"등록"** 버튼을 탭하여 댓글을 게시합니다.

[스크린샷: 댓글 입력란과 등록 버튼]

## 대댓글 (답글)

특정 댓글에 답글을 달 수 있습니다:

1. 답글을 달 댓글의 **"답글"** 버튼을 탭합니다.
2. 댓글 입력란이 대댓글 모드로 전환됩니다.
3. 내용을 입력하고 **"등록"**을 탭합니다.
4. 대댓글 모드를 취소하려면 **"취소"** 버튼을 탭합니다.

[스크린샷: 대댓글 작성 모드]

## 댓글 수정

1. 본인이 작성한 댓글의 **"수정"** 버튼을 탭합니다.
2. 댓글 내용이 편집 가능한 상태로 전환됩니다.
3. 내용을 수정한 후 **"저장"** 버튼을 탭합니다.

> **주의:** 본인이 작성한 댓글만 수정할 수 있습니다.

## 댓글 삭제

1. 본인이 작성한 댓글의 **"삭제"** 버튼을 탭합니다.
2. "댓글을 삭제하시겠습니까?" 확인 다이얼로그가 표시됩니다.
3. **"삭제"**를 탭하면 댓글이 삭제됩니다.
4. 삭제된 댓글은 **"삭제된 댓글입니다"**로 표시됩니다.

[스크린샷: 댓글 삭제 확인 다이얼로그]

> **팁:** 대댓글이 있는 댓글을 삭제해도 대댓글은 유지됩니다. 원래 댓글 위치에 "삭제된 댓글입니다"가 표시됩니다.

## 익명 기능

글과 댓글 모두 익명으로 작성할 수 있습니다.

### 글의 익명 작성

- 글쓰기 폼에서 **"익명"** 옵션을 체크하면 작성자 이름 대신 "익명"으로 표시됩니다.

### 댓글의 익명 작성

- 댓글 입력 시 **"익명"** 옵션을 선택할 수 있습니다.
- 같은 글에서 한번 익명으로 댓글을 달면, 이후 해당 글에서 작성하는 댓글에 동일한 익명 설정이 유지됩니다.

### 익명 기능 안내

- 익명 글/댓글의 작성자는 다른 사용자에게 공개되지 않습니다.
- 익명 글에 대해 댓글을 달 때도 익명 여부를 선택할 수 있습니다.
- 수정/삭제는 익명 여부와 관계없이 **본인 작성분만** 가능합니다.

[스크린샷: 댓글 작성 시 익명 체크박스]

> **팁:** 익명 기능은 자유로운 의견 개진을 위한 것이나, 병원 내 건전한 소통 문화를 위해 적절하게 사용해주세요.
