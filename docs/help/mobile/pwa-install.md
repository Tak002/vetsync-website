# 앱처럼 설치하기 (PWA)

VetSync는 PWA(Progressive Web App)를 지원하여, 웹 브라우저에서 접속하는 웹사이트를 네이티브 앱처럼 홈 화면에 설치할 수 있습니다.

## PWA란

- 별도의 앱스토어 다운로드 없이, 브라우저에서 바로 설치
- 홈 화면에 앱 아이콘이 추가되어 한 번의 탭으로 실행
- 독립 앱처럼 실행되어 브라우저 주소창이 표시되지 않음
- 서버 업데이트 시 앱이 자동으로 최신 버전으로 갱신

## iOS Safari에서 설치

iPhone 또는 iPad의 Safari 브라우저에서 설치합니다.

1. **Safari**로 VetSync 웹사이트에 접속합니다.
2. 하단 메뉴바의 **공유 버튼** (사각형에 위쪽 화살표 아이콘)을 탭합니다.
3. 메뉴를 스크롤하여 **"홈 화면에 추가"**를 선택합니다.
4. 앱 이름을 확인하고 **"추가"**를 탭합니다.
5. 홈 화면에 VetSync 아이콘이 추가됩니다.

> **주의:** iOS에서는 반드시 **Safari**를 사용해야 합니다. Chrome, Firefox 등 다른 브라우저에서는 홈 화면 추가 기능이 지원되지 않습니다.

> **팁:** VetSync에 처음 접속하면 상단에 설치 안내 배너가 자동으로 표시됩니다. 배너의 안내에 따라 설치하면 더 간편합니다. 배너를 닫으면 14일 후 다시 표시됩니다.

## Android Chrome에서 설치

Android 기기의 Chrome 브라우저에서 설치합니다.

1. **Chrome**으로 VetSync 웹사이트에 접속합니다.
2. 다음 중 하나의 방법으로 설치합니다:
   - 주소창 근처에 **설치 아이콘**이 표시되면 탭합니다.
   - 또는 Chrome 메뉴(점 세 개) → **"앱 설치"**를 선택합니다.
3. **"설치"** 확인 버튼을 탭합니다.
4. 홈 화면에 VetSync 앱 아이콘이 추가됩니다.

## 설치 후 사용

### 앱 실행

- 홈 화면의 VetSync 아이콘을 탭하면 독립 앱 모드로 실행됩니다.
- 브라우저 주소창, 탭, 메뉴바 없이 전체 화면으로 표시됩니다.

### 자동 업데이트

- VetSync 서버에 업데이트가 배포되면 앱이 자동으로 최신 버전을 확인합니다.
- 약 60초마다 업데이트를 체크하여, 새 버전이 있으면 자동으로 반영됩니다.
- 별도의 수동 업데이트가 필요 없습니다.

### PWA 모드의 장점

- **화면 항상 켜기** 기능이 PWA 모드에서 가장 안정적으로 동작합니다.
- 알림(Web Push)을 수신할 수 있습니다.
- 브라우저 탭과 독립적으로 실행되어 실수로 탭을 닫을 위험이 없습니다.

> **팁:** 진료 현장에서 태블릿이나 스마트폰으로 VetSync를 사용한다면, PWA로 설치하여 사용하는 것을 권장합니다.
