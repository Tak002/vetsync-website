# 빠른입력 사용법

빠른입력은 모바일에서 활력징후(체온, 심박수, 호흡수, 혈압)를 빠르게 기록하는 전용 기능입니다. 차트를 열지 않고도 간편하게 바이탈 사인을 입력할 수 있습니다.

## 빠른입력이란

- 활력징후 4가지를 한 화면에서 빠르게 입력
- 현재 시간 슬롯에 자동으로 저장
- 차트 상세 화면을 열 필요 없이 간편한 기록 가능

## 접근 방법

하단 네비게이션의 **3번째 탭**(번개 아이콘)을 탭합니다.

[스크린샷: 하단 네비게이션에서 빠른입력 탭 위치]

## 차트 선택

빠른입력을 사용하려면 먼저 대상 차트를 지정해야 합니다.

### 모바일

1. 화면 우측 상단의 **차트 선택 버튼**을 탭합니다.
2. 오늘 날짜의 차트 목록이 표시됩니다.
3. 원하는 환자의 차트를 선택합니다.
4. 선택한 차트의 환자 이름이 상단에 표시됩니다.

### PC

1. 입력값을 먼저 작성합니다.
2. **"다음: 차트 선택"** 버튼을 클릭합니다.
3. 오늘 차트 목록에서 대상 차트를 선택합니다.

[스크린샷: 오늘 차트 목록에서 환자 선택하는 모습]

> **팁:** 차트 상세 화면에서 빠른입력으로 이동하면 해당 차트가 자동으로 선택됩니다.

## 입력 타입별 안내

빠른입력 화면에는 4가지 바이탈 사인 입력 카드가 표시됩니다. 필요한 항목만 입력하면 됩니다.

### 체온

- **숫자 키패드**로 체온을 직접 입력합니다.
- 소수점 입력을 지원합니다 (예: 38.5).

[스크린샷: 체온 키패드 UI]

### 심박수

- **화면 탭 인터페이스**로 BPM을 측정합니다.
- 심박에 맞춰 화면을 탭하면 자동으로 BPM이 계산됩니다.
- 직접 숫자를 입력할 수도 있습니다.

[스크린샷: 심박수 탭 측정 UI]

### 호흡수

- 호흡수를 숫자로 입력합니다.

[스크린샷: 호흡수 입력 UI]

### 혈압

- **수축압**과 **이완압**을 동시에 입력합니다.
- 슬래시(/)로 구분하여 표시됩니다 (예: 120/80).

[스크린샷: 혈압 입력 UI]

## 기존 값 덮어쓰기

해당 시간 슬롯에 이미 값이 입력되어 있는 경우:

1. 저장 시 **"덮어쓰기 확인"** 다이얼로그가 표시됩니다.
2. 충돌하는 항목 목록이 표시됩니다 (예: "체온, 심박수에 이미 값이 있습니다").
3. **"덮어쓰기"**를 선택하면 기존 값을 새 값으로 교체합니다.
4. **"취소"**를 선택하면 저장하지 않고 돌아갑니다.

[스크린샷: 덮어쓰기 확인 다이얼로그]

> **주의:** 덮어쓰기하면 기존 값은 복구할 수 없습니다.

## 저장 및 완료

1. 원하는 항목을 모두 입력합니다.
2. **"저장"** 버튼을 탭합니다.
3. 저장이 완료되면 **"빠른 입력 저장이 완료되었습니다"** 메시지가 표시되며, 해당 차트 상세 화면으로 자동 이동합니다.

> **팁:** 현재 시각의 시간 슬롯에 자동 저장됩니다. 화면 상단에 현재 시간 슬롯(예: "14시")이 표시되어 있으니 확인해주세요.

> **팁:** 일부 항목만 입력해도 저장할 수 있습니다. 체온만 입력하고 저장하면 체온만 기록됩니다.
