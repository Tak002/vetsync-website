# 화면 항상 켜기

진료 중 기기 화면이 자동으로 꺼지지 않도록 방지하는 기능입니다. 수액 모니터링, 지속 관찰 등 장시간 화면을 확인해야 하는 상황에서 유용합니다.

## 기능 설명

- **화면 항상 켜기**를 활성화하면 기기의 자동 잠금(화면 꺼짐)이 비활성화됩니다.
- 브라우저의 Screen Wake Lock API를 사용합니다.
- 별도의 앱 설치 없이 웹 브라우저에서 동작합니다.

## 설정 방법

1. **설정** 페이지로 이동합니다.
   - PC: 상단 네비게이션의 설정 아이콘
   - 모바일: 하단 네비게이션 **더보기** → **설정**
2. **"화면 항상 켜기"** 토글을 **ON**으로 전환합니다.
3. 설정이 즉시 적용됩니다.

[스크린샷: 설정 페이지의 화면 항상 켜기 토글]

> **팁:** 이 설정은 기기에 저장되므로, 다음에 접속할 때도 유지됩니다.

## 동작 방식

### 활성화 시

- 기기 화면이 자동으로 꺼지지 않습니다.
- VetSync를 사용하는 동안 계속 화면이 켜진 상태를 유지합니다.

### 탭 전환 시 자동 재획득

- 다른 브라우저 탭이나 앱으로 전환하면 화면 잠금 방지가 일시 해제됩니다.
- VetSync 탭으로 다시 돌아오면 **자동으로 재획득**합니다.
- 별도의 조작 없이 원래 상태로 복원됩니다.

### 비활성화 시

- 토글을 OFF로 전환하면 기기의 기본 자동 잠금 설정이 적용됩니다.
- 설정된 시간이 지나면 화면이 자동으로 꺼집니다.

## 주의사항

### 배터리 소모

- 화면이 계속 켜져 있으므로 **배터리 소모가 증가**합니다.
- 장시간 사용 시 충전기를 연결하는 것을 권장합니다.

### 브라우저 호환성

- 최신 Chrome, Edge, Safari(iOS 16.4+)에서 지원됩니다.
- 지원하지 않는 브라우저에서는 토글이 동작하지 않을 수 있습니다.

### PWA 모드에서 가장 안정적

- PWA(홈 화면에 추가)로 설치하여 사용하면 화면 항상 켜기가 가장 안정적으로 동작합니다.
- 일반 브라우저 탭에서는 다른 탭으로 전환 시 일시적으로 해제될 수 있습니다.
- PWA 설치 방법은 [앱처럼 설치하기](./pwa-install.md)를 참고하세요.

> **팁:** 태블릿을 거치대에 고정하고 모니터링용으로 사용하는 경우, "화면 항상 켜기"와 충전기를 함께 사용하면 항시 환자 상태를 확인할 수 있습니다.
