# 모바일 화면 구성

VetSync는 기기 유형에 따라 최적화된 화면 레이아웃을 제공합니다. 모바일(스마트폰)과 태블릿에서는 터치에 최적화된 하단 네비게이션을 사용합니다.

## 하단 5탭 네비게이션

모바일 및 태블릿에서는 화면 하단에 5개 탭으로 구성된 네비게이션 바가 표시됩니다.

| 순서 | 탭 | 아이콘 | 기능 |
|------|-----|--------|------|
| 1 | **차트 목록** | 차트 | 오늘 차트 리스트, 대시보드 뷰 |
| 2 | **업무** | 체크리스트 | 마킹된 업무(셀) 목록 |
| 3 | **빠른입력** | 번개 | 활력징후 빠른 기록 |
| 4 | **차트검색** | 돋보기 | 과거 차트 검색 |
| 5 | **더보기** | 메뉴 | 게시판, 환자/보호자, 설정, 로그아웃 |

[스크린샷: 하단 5탭 네비게이션 바]

### 차트 목록 탭

- 오늘 날짜의 입원/진료 차트가 목록으로 표시됩니다.
- 차트를 탭하면 차트 상세 화면으로 이동합니다.

### 업무 탭

- 마킹(체크)된 업무 셀을 시간순으로 정리하여 보여줍니다.
- 담당 업무를 빠르게 확인하고 완료 처리할 수 있습니다.

### 빠른입력 탭

- 체온, 심박수, 호흡수, 혈압을 빠르게 입력하는 전용 화면입니다.
- 자세한 사용법은 [빠른입력 사용법](./quick-input.md)을 참고하세요.

### 차트검색 탭

- 날짜, 환자명 등으로 과거 차트를 검색합니다.
- 퇴원한 환자의 차트 기록을 찾을 때 유용합니다.

### 더보기 탭

더보기 페이지에는 다음 메뉴가 제공됩니다:

- **사용자 정보**: 현재 로그인한 사용자의 이름, 역할, 이메일
- **게시판**: 병원 내 자유 게시판
- **환자/보호자**: 환자 및 보호자 관리
- **설정**: 앱 설정
- **로그아웃**: 현재 계정에서 로그아웃

[스크린샷: 더보기 페이지 메뉴 목록]

## 태블릿 모드 (768px 이상)

화면 너비가 768px 이상인 태블릿에서는 추가 기능을 사용할 수 있습니다:

- **넓은 화면 활용**: 차트 상세에서 더 많은 정보를 한 화면에 표시
- **뷰 모드 전환**: 차트 뷰에서 shrink(축소) 모드와 scroll(스크롤) 모드를 전환할 수 있습니다
  - **shrink 모드**: 모든 시간 슬롯을 화면에 맞게 축소 표시
  - **scroll 모드**: 원래 크기로 표시하며 좌우 스크롤
- 하단 네비게이션은 동일하게 유지됩니다.

[스크린샷: 태블릿에서 shrink/scroll 뷰 전환 버튼]

## PC와의 차이점

| 항목 | PC (비터치 기기) | 모바일/태블릿 (터치 기기) |
|------|-----------------|------------------------|
| 네비게이션 | 상단 헤더 (로고, 프로필, 설정) + 상단 탭 | 하단 5탭 네비게이션 |
| 차트 뷰 | shrink 모드 고정 | shrink/scroll 전환 (태블릿), scroll 고정 (모바일) |
| 대시보드 | 격자 뷰 (열 수 수동 지정) | 리스트 뷰 |
| 헤더 | 표시 | 없음 |
| 더보기 페이지 | 없음 (`/`로 리다이렉트) | 게시판/환자/설정 메뉴 허브 |
| 계산기 패널 | 차트 옆 사이드 패널 | 바텀 시트 |

> **팁:** VetSync는 터치 입력 장치(pointer: coarse)를 감지하여 자동으로 모바일/태블릿 레이아웃을 적용합니다. 화면 크기가 아닌 **입력 방식** 기준으로 레이아웃이 결정됩니다.
